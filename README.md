# 🚀 Microservices Fullstack App

## 📌 Overview

A full-stack microservices-based application built using React, Node.js, and Docker.

## 🧱 Architecture

* React Frontend
* Auth Service (Node.js)
* Product Service (Node.js)
* Order Service (Node.js)
* Docker Compose for orchestration

## ⚙️ Tech Stack

* React.js
* Node.js + Express
* Docker & Docker Compose
* REST APIs

## 🚀 How to Run

### 1. Start Backend

```bash
docker compose up --build
```

### 2. Start Frontend

```bash
cd frontend
npm install
npm start
```

### 3. Open App

http://localhost:3000

## 🔥 Features

* User Login (Auth Service)
* Product Listing (Product Service)
* Order Placement (Order Service)
* Microservices Architecture
* Dockerized Backend
